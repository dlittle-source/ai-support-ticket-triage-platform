# Sample Support Tickets

Use these sample tickets to test the AI Support Ticket Triage Starter Kit.

---

## Critical Severity

### Ticket 1
Production API is returning 503 errors after the latest deployment. Users cannot access the application, and the frontend is showing gateway timeout errors.

### Ticket 2
Payment processing is failing for all customers. Checkout requests are timing out, and revenue-impacting transactions are not completing.

---

## High Severity

### Ticket 3
Users are unable to log in after the latest authentication update. The login page loads, but valid credentials return an unexpected server error.

### Ticket 4
Database CPU usage is above 90%, and several API endpoints are responding slowly during peak traffic.

---

## Medium Severity

### Ticket 5
The dashboard metrics are delayed by several minutes. Data eventually loads, but users are reporting stale status information.

### Ticket 6
A scheduled background job failed overnight. Reports were not generated, but the main application is still available.

---

## Low Severity

### Ticket 7
A button label on the settings page is confusing and should be renamed for better clarity.

### Ticket 8
The help text under the search field should include an example query to improve user experience.
